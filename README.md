# APUS IVA Expansion

This is a small KSP mod that adds proper IVAs to parts from the APUS mod and makes them fully compatible with FreeIVA

## Notes on License

As there was no License provided for APUS, I am required to consider all rights reserved. Nontheless, this expansion contains a custom IVA for the Entrance part of APUS, which I had to stitch together from other parts.
Since I have modified parts to create a new part, this would be an invalid redistribuition of APUS owned assets. For this reason, if you have any problems with this file being contained in this mod, please contact me and let me know to remove it.

### Disclaimer

- This does not include Textures... (knock yourself out if you want to add them yourself)
- These IVAs do not have the best quality, or colliders, so for some doors, you'll have to crouch to get through them
- There are some visual artifacts, but I can't be bothered to fix them right now
- The docking passthrough inside Cargo bays is funky, if it doesnt let you pass through, try snapping the other node to the cargo bay

### Dependencies
This mod DEPENDS on [NearFutureProps](https://forum.kerbalspaceprogram.com/topic/166941-112x-near-future-props-prop-assets-for-iva-developers-august-13th/)   
This mod DEPENDS on [FreeIVA](https://forum.kerbalspaceprogram.com/topic/210548-112-freeiva/)

### Installation

Just put this folder into your GameData folder. Thats it
